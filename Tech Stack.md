# Tech Stack

## Application
- Base - NextJS (App Router)
- DI - Inversify
- UI - Shadcn
- ORM - Drizzle
- API - RESTful API
- User Authentication - Form + Cookie
- API Authentication - JWT
- Caching - NextJs built-in cache, fetch catching, unstable-cache
- Migration - Drizzle
- Form Validation - Zod Schema
## Database
MySql
